🧠 Répétition espacée

L'algorithme SRS priorise les scénarios à revoir. Plus tu maîtrises, moins souvent tu les revois. Style Anki, sans effort.

📋 Historique et analyse

Consulte tes sessions passées, ton taux de réussite par type de question, et reprends une session interrompue où tu t'es arrêté.

📱 Pensé pour mobile

Interface optimisée pour l'utilisation dans la voiture. Navigation par swipe, boutons accessibles, pas de zoom intempestif sur iOS.
